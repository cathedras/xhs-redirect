# 修复二维码内容 - 使用真实地址

## 问题
用户反馈：二维码内容错误，扫描后显示的是短链地址而非真实地址。

正确地址应为：
```
https://www.xiaohongshu.com/miniapp/qrcode?background_transparent=true&disableNativeLoading=yes&fullscreen=true&i=6945577745f0d800019e4a33&k=16&s=SKB000003&p=pages%2Fcompose%2Findex%3FxhsMpBizQuery%3Dscene%253DSKB000003
```

## 修复内容

### 1. 微信小程序跳转页 (weapp-redirect.html)
- 添加短链解析逻辑：调用 `/api/resolve` 接口获取真实地址
- 二维码内容改为使用解析后的 `realUrl`
- 复制链接功能也使用真实地址

### 2. 二维码生成器 (qr-generator.html)
- 添加异步解析功能：输入短链时自动调用 API 解析
- 二维码内容使用解析后的真实地址
- 保持原有 Logo 上传和下载功能

## 技术实现

```javascript
// 如果是短链，先解析获取真实地址
if (url.includes('xhslink.com')) {
    const response = await fetch('/api/resolve?url=' + encodeURIComponent(url));
    const data = await response.json();
    if (data.success && data.realUrl) {
        qrContent = data.realUrl;
    }
}
```

# 小红书短链跳转服务 - 二维码和微信小程序页面

## 目标
为已部署的 Vercel 服务添加：
1. 带自定义 Logo 的二维码生成器
2. 微信小程序跳转页面

## 已完成工作

### 1. 二维码生成器 (qr-generator.html)
- 支持输入任意 URL 生成二维码
- 支持上传自定义 Logo 并居中显示
- 提供下载功能（PNG格式）
- 默认预填服务地址

**访问地址：**
```
https://xhs-redirect.vercel.app/qr
```

### 2. 微信小程序跳转页面 (weapp-redirect.html)
- 自动识别微信环境并给出提示
- 居中显示二维码，支持长按识别
- 提供"打开小红书 APP"按钮
- 提供"复制链接"功能
- 适配 iOS/Android URL Scheme

**访问地址：**
```
https://xhs-redirect.vercel.app/weapp
```

### 3. 服务端更新
- 添加 `/qr` 路由指向二维码生成器
- 添加 `/weapp` 路由指向小程序跳转页

### 4. 代码已推送
- 已提交到 GitHub 仓库
- Vercel 会自动重新部署

## 使用说明

### 生成带 Logo 的二维码
1. 访问 `https://xhs-redirect.vercel.app/qr`
2. 输入跳转链接（默认已填好）
3. 上传 Logo 图片（可选）
4. 点击"生成二维码"
5. 下载 PNG 图片

### 微信小程序使用
1. 访问 `https://xhs-redirect.vercel.app/weapp?param=短链URL`
2. 页面显示二维码和跳转按钮
3. 微信内会提示"在浏览器打开"
4. 浏览器中会自动尝试打开小红书 APP

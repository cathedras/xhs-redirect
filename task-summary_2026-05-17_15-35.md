# 优化小红书跳转按钮功能

## 目标
模仿"薯客宝"的跳转按钮功能，实现点击后直接跳转小红书APP

## 优化内容

### 1. 跳转逻辑优化 (weapp-redirect.html)
- 使用多种 URL Scheme 尝试打开APP：
  - `xhsdiscover://`
  - `xhs://`
  - `xiaohongshu://`
- 使用 iframe 方式尝试打开（兼容更多浏览器）
- 如果APP未安装，自动跳转到真实链接

### 2. 按钮样式优化
- 圆角从 12px 改为 25px（更像薯客宝）
- 添加渐变动画效果（pulse）
- 添加光泽扫过效果（hover）
- 添加阴影效果
- 按钮文字改为"📱 打开小红书"

### 3. 跳转流程
```
用户点击按钮
    ↓
检测是否微信环境 → 是 → 显示提示
    ↓ 否
检测 iOS/Android
    ↓
iOS: 依次尝试 xhsdiscover://, xhs://, xiaohongshu://
Android: 使用 intent:// 方式
    ↓
成功打开APP → 结束
    ↓ 失败（1.5秒内未跳转）
自动跳转到真实链接
```

## 技术实现

```javascript
function tryOpenApp(schemes) {
    let index = 0;
    
    function tryNext() {
        if (index >= schemes.length) {
            window.location.href = realUrl;
            return;
        }
        
        const scheme = schemes[index++];
        const iframe = document.createElement('iframe');
        iframe.style.display = 'none';
        iframe.src = scheme;
        document.body.appendChild(iframe);
        
        setTimeout(() => {
            document.body.removeChild(iframe);
            if (!document.hidden) {
                tryNext();
            }
        }, 500);
    }
    
    tryNext();
}
```

# 实现微信开放标签跳转小程序

## 目标
在微信内直接跳转小红书小程序，无需扫码

## 实现方案

### 1. 微信开放标签 (wx-open-launch-weapp)
在微信环境中使用官方提供的开放标签：

```html
<wx-open-launch-weapp
    id="launch-btn"
    username="gh_d3c1c85e5a7a" 
    path="pages/compose/index?scene=SKB000003">
    <template>
        <button>📱 打开小红书</button>
    </template>
</wx-open-launch-weapp>
```

### 2. 配置微信JS-SDK
```javascript
wx.config({
    debug: false,
    appId: 'your-app-id',
    timestamp: 1234567890,
    nonceStr: 'random-string',
    signature: 'generated-signature',
    jsApiList: [],
    openTagList: ['wx-open-launch-weapp']
});
```

### 3. 环境判断
- **微信内**：显示开放标签按钮
- **非微信**：显示普通按钮（scheme跳转）
- **配置失败**：降级显示普通按钮

## 重要参数

### username（小程序原始ID）
- 格式：`gh_xxxxxxxx`
- 需要替换为小红书小程序的真实原始ID
- 获取方式：小程序后台 → 设置 → 原始ID

### path（跳转路径）
- 格式：`pages/xxx/xxx?param=value`
- 从链接参数中提取：`p=pages%2Fcompose%2Findex`
- 解码后：`pages/compose/index`

## 限制条件

1. **需要公众号**
   - 必须拥有已认证的微信公众号
   - 用于配置JS-SDK权限

2. **需要配置安全域名**
   - 在公众号后台配置JS接口安全域名
   - 添加你的跳转服务域名

3. **需要生成签名**
   - 服务端生成微信JS-SDK签名
   - 需要appId、timestamp、nonceStr、url

4. **小红书原始ID**
   - 需要获取小红书小程序的原始ID
   - 格式为 `gh_xxxxxxxx`
